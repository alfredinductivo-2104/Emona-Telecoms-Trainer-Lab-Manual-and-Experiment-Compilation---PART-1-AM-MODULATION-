# SETTING UP AN OSCILLOSCOPE

---

## INTRODUCTION
An Oscilloscope is an electronic measuring device used to analyze and observe electrical signals. It allows the user to see the signal shape, frequency, and amplitude. Additionally, this device allows the user to measure different signal parameters such as the period (T), frequency (f), peak voltage (Vp), peak-to-peak voltage (Vpp), and time intervals. Oscilloscopes are widely used in electronics laboratories for testing, troubleshooting, and verifying the performance of electronic circuits and systems.

## THINGS YOU NEED TO KNOW:
- The amplitude of a signal is its physical size and is measured in volts (V). It is usually measured either from the middle of the waveform to the top (called the peak voltage) or from the bottom to the top (called the peak-to-peak voltage).
- The period of a signal is the time taken to complete one cycle and is measured in seconds (s). When the period is small, the period is expressed in milli seconds (ms) and even micro seconds (us).
- The frequency of a signal is the number of cycles every second and is measured in hertz (Hz). When there are many cycles per second, the frequency is expressed in kilo hertz (kHz) and even mega hertz (MHz)

### PROCEDURE OF SETTING UP AN OSCILLOSCOPE:
1.	Gather all necessary equipment for the activity, ensuring that everything is easily accessible and ready for use.
2.	Verify that all equipment is undamaged and functioning properly. Ensure each piece is correctly calibrated.
3.	In General Controls, Set the Intensity Control to about three quarters and Set the Mode control to the CH1 position
4.	In Vertical Control, Set the Input Coupling control for both channels to the AC position. Then, Set the Vertical Attenuation control for both channels to the 1 V/div position. After that, Set the Vertical Attenuation Calibration control for both channels to the detent (locked) position. Lastly, Set the Vertical Position control for both channels to about the middle of their travel 
5.	While in the, Horizontal controls: Set the Horizontal Timebase control to the 0.5ms/div position. Then, Set the Horizontal Timebase Calibration control to the detent (locked) position and lastly, Set the Horizontal Position control to about the middle of its travel. 
6.	In the Triggering controls, Set the Sweep Mode control to the AUTO position. Then, Set the Trigger Level control to the detent (locked) position. If it doesn't have a decent position, set it to about the middle of its travel. Then, Set the Trigger Source control to the CHI (or INT) position.
